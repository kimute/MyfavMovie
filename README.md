# FavMovie

a favorite Movie Search Web created by React & ES6

## Menu

- [v] Home
  - Now Playing movie
  - Popular movie
  - Upcomming movie
- [v] Movie
  - My favorate movie (over 8 rating)
  - My unfavorat movie (Horror movies)
- [v] Search
- [v] Detail

## Packages

- react-router-dom
- styled-components
- styled-reset(for reset css)
- Axios (for get API )
- prop-types
- React-player(for Youtube)
